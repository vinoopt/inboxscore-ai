Authentication Rates (SPF, DKIM, DMARC)
Basic SPF and DKIM pass rates from Google's API. Google's web UI shows alignment-aware rates (a stricter measure) — those aren't exposed via API. Learn more
SPF / DKIM pass vs alignment
Basic SPF pass = sending IP authorised by the Return-Path domain. SPF alignment (what Google's UI shows) = pass and Return-Path matches the visible From: domain. Mail relayed through ESPs often passes basic SPF but fails alignment. DMARC inherently requires alignment, so its API value matches the UI.

What to do if you see persistent yellow or red
  • Persistent Throttled (yellow): warm up the IP slower, audit complaint rate, clean inactive recipients from your lists.
  • Persistent Spam-filtered (red): Microsoft is rejecting your mail. File an unblock / mitigation request at senders.microsoft.com.
  • No-volume rows: normal if the IP doesn't typically send to Outlook / Hotmail. Ignore unless you expected volume there.
